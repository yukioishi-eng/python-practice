# python-practice

## 概要
大学1年次・エンジニア志望として、Pythonの基礎力を身につけるための学習用リポジトリです。  
基礎文法の理解だけでなく、「コードを書いて動かす」ことを重視し、GitHub上で成果を可視化しています。

## 学習内容
- Python基礎文法（if / for / list / dict）
- 条件分岐や繰り返し処理の理解
- 小さな処理を自力で書く演習

## 実績・サンプル
- リストや辞書を使った簡易成績管理プログラムを作成
- if / for を使った条件処理・繰り返し処理の演習コードを複数実装
- GitHubに学習ログとして継続的にコードを記録

## 工夫したこと
- コードの再利用性を意識して関数化
- 入力エラーや例外処理を加えて実用性を意識
- GitHubでこまめにコミットし、作業履歴を可視化

## ファイル構成
- main.py  
  学習ログとして、学んだことや試したコードのまとめを記録

- basics/  
  Python基礎演習コードをまとめたフォルダ  
  例: 条件分岐や繰り返し処理の演習コードを収録

## 学習方針
- 毎回必ずコードを書く
- 学習した内容をGitHubに記録する
- 後から見て理解できるコードを意識する

## 今後挑戦したいこと
- 関数・クラスの基礎理解の応用
- アルゴリズム的思考の習得
- Pythonで簡単なWebスクレイピングツールやミニアプリの作成
- データ可視化アプリ（Matplotlib / Pandas使用）

## 使用技術
- Python
- Git / GitHub
